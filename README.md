# 🚀 Wazuh - Plateforme de Motivation

## 📌 Description
Wazuh est une plateforme web interactive de motivation quotidienne. Elle génère des citations inspirantes, permet de choisir une humeur, sauvegarde les utilisateurs et conserve un historique des citations.

---

## 🎯 Fonctionnalités

- Génération de citations motivantes dynamiques
- Choix de l’humeur (motivation, inspiration, confiance)
- Système d’inscription utilisateur
- Sauvegarde locale (localStorage)
- Historique des citations
- Filtrage des citations par catégorie
- Mode sombre / clair
- Interface responsive

---

## 🛠️ Technologies utilisées

- HTML5
- CSS3 (variables + responsive design)
- JavaScript (DOM, events, localStorage)
